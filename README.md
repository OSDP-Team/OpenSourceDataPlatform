# OSDP Projekt

Dieses Repository enthält die Komponenten und Konfigurationen für das OSDP-Projekt.

## Beschreibung

Das OSDP ist eine Daten Plattform die ausschließlich mit Open-Source Technologien umgesetzt wird. 
Die Open-Source Operatoren werden mit Stackable auf einem Kubernetes Cluster aufgesetzt und konfiguriert. 
Zum Betreiben des Clusters wird Microsoft Azure verwendet.

## Operatoren 

### Airflow

Operator für Orchestrierung und Automatisierung 

### Keyckloak

Authentifizierungsoperator

### MinIO

S3 kompatibler Objektspeicher für Cloud-Native Umgebungen

### PostgreSQL

Datenbank

### Spark

Operator für Datenverarbeitung

### Superset 

Visualisierungsoperator

## Ziel 

Die OSDP soll zum aktuellen Zeitpunkt das Ziel verfolgen, Daten von Heizkraftwerken zu verarbeiten, zu speichern und zu visualisieren. Später sollen eventuell Möglichkeiten eingebunden werden, um Live-Daten zu verarbeiten.

## Inhalt

Das Repository umfässt alle Cloud Konfigurationen, sowie scripts, Dokumentationen und lokale Demos.
